Grillas: Imagina las grillas como una especie de estructura que divide la página en filas y columnas, creando secciones donde podemos colocar nuestro contenido. Esto nos ayuda a organizar todo de manera ordenada, como dividir una hoja de papel en secciones para escribir diferentes cosas.
Columnas: Dentro de esas filas, tenemos columnas que actúan como los espacios donde colocamos nuestro contenido. Piensa en ellas como bloques que ocupan cierta cantidad de espacio en la página. Con las columnas, podemos decidir cuánto espacio ocupa cada parte de nuestro contenido, lo que nos permite diseñar la página de manera flexible y adaptativa.
Media Queries: Son como instrucciones especiales que le damos a nuestra página para que se adapte a diferentes dispositivos y tamaños de pantalla. Con las media queries, podemos decirle a la página cómo debe comportarse en teléfonos móviles, tabletas o computadoras de escritorio, asegurándonos de que se vea bien en cualquier dispositivo.
Container: El contenedor es como el marco principal de nuestra página, es decir, el espacio donde ocurre toda la magia. Todo el contenido importante debe estar dentro de este contenedor para mantenerlo organizado y centrado en la pantalla.
Row y Col: Las filas y columnas trabajan juntas para organizar el contenido dentro del contenedor. Las filas agrupan las columnas y aseguran que estén alineadas correctamente, creando una estructura visualmente atractiva y fácil de leer para los usuarios.
